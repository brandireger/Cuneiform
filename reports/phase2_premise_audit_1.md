# Phase 2 premise audit 1

**[INTERNAL PREMISE AUDIT — not for citation]**

Triggered after P2-A, P2-D, and P2-B under `SANDBOX_RULES.md` rule 13.

## Evidence now on record

- P2-A: TLHdig does not encode a member-specific within-line fracture
  column, so the proposed handed-over-truth seam-localization test has no
  valid target.
- P2-D: 77/182 canonical dev relation pairs are indirect or expanded from
  shared-line attribution, but the frozen BM25 hard set is not enriched for
  those weaker bases at its query unit.
- P2-B: symbolic layout and damage structure are available, while contour,
  dimensions, surface geometry, clay properties, imagery/3D, and graphetic
  hand are absent.

## Premise decision

Direct pairwise textual scoring is not an adequate general formulation for
no-overlap physical joining in this corpus. More reranking, seam scoring,
or graph constraints cannot manufacture the missing physical evidence.
Tier-A output must be allowed to abstain as `insufficient encoded evidence`.

The project should retain:

- textual-affinity and duplicate/parallel retrieval;
- symbolic layout compatibility and contradiction checks;
- direct/indirect/shared-line relation stratification;
- physical-join candidate packets that expose missing evidence.

## Next justified probe

Proceed to P2-E as a feasibility census before any inference run: measure how
many dev direct joins—especially tier A/B—have a third same-composition
witness with enough attested overlap to mediate a bridge. This asks whether
the corpus's duplicate abundance can supply missing textual context without
pretending it supplies physical geometry. If coverage is too low, stop P2-E
before modeling.

P2-C remains a secondary diagnostic, not the next architecture: its frozen
model's measured continuation horizon is already short, and it still asks
text alone to bridge a class defined by no shared text.
